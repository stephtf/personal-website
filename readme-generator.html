readme generator

about this project

Have you ever spent way too much time writing a readme? Yes, you can save valuable time and get back to coding by using this readme generator! This application makes it super easy to just answer a few basic questions about your project that will then be automatically compiled into a pre-formatted readme. It runs in node.js where you will be prompted to answer the questions. When you finish, the readme is generated into the project file.

my role

I created this readme generator as an assignment in my coding bootcamp. I wrote all of the functions to prompt user input and to write the file.
