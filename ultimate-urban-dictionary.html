the ultimate urban dictionary

about this project

I collaborated with two other team members to create The Ultimate Urban Dictionary, an application that allows users to search for trendy and/or slang words to find their definitions as well as a corresponding gif. This was a super fun project to work on! We wanted to build something that would help our target user: anyone who wants to learn some hip lingo!

my role

This was truly a collaborative effort from start to finish. I helped to define the project and its aesthetic as well as its functionality from the very beginning. I was also heavily involved in writing the code during the development phase. In particular, I was responsible for the CSS stylings, as well as adding functionality to the app via Javascript.
